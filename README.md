# Contact Form with LocalStorage (Task 3)

This project is part of my **Full Stack Web Development Internship – Task 3**.  
The goal of this task was to add **JavaScript interactivity** to a multi-page website by implementing a contact form with validation and LocalStorage functionality.

## 📌 Project Overview
The website includes a **Contact Form** where users can submit their details (Name, Email, Message).  
The submitted data is validated using JavaScript and then stored in the browser using **LocalStorage**.  
All saved messages are displayed dynamically on a **Submissions page**.

## 🚀 Features
- Contact Form with **Name, Email, and Message fields**
- **Client-side validation** to ensure no empty fields
- **LocalStorage integration** to store form data
- **Submissions page** to view saved messages
- **Clear All button** to remove stored messages
- Responsive and modern UI design

## 🛠️ Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (DOM Manipulation)**
- **LocalStorage API**

## 📂 Project Structure
```
project-folder
│
├── index.html        # Home page
├── contact.html      # Contact form page
├── submissions.html  # Display saved messages
├── style.css         # Styling
└── script.js         # JavaScript logic
```

## ⚙️ How It Works
1. User fills the contact form.
2. JavaScript validates the input fields.
3. If valid, the data is stored in **LocalStorage**.
4. The **Submissions page** fetches and displays saved messages dynamically.
